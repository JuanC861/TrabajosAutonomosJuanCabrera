# TrabajosAutonomosJuanCabrera

Pong v.1 – Aprendizaje Autónomo 2

Autor: Juan Cabrera

Python + pygame-ce

Descripción: Juego tipo Pong hecho en Python usando pygame-ce, aplicando lógica de programación (bucles, condicionales, funciones, eventos y colisiones).

Funcionalidades: 

- “Clic para empezar”
- Rebotes y colisiones con paletas
- Vidas (3) por jugador con corazones
- Pantalla final con Reiniciar y Salir

Controles:

- Izquierda: W / S
- Derecha: ↑ / ↓


Requisitos:

- Instalar pygame-ce: pip install pygame-ce

Ejecutar: 

- python "Juan Cabrera Pong - Aprendizaje Autonomo 3 - Evaluacion con el Docente"

Archivos (Necesarios para que se vea el corazón):

- Juan Cabrera Pong - Aprendizaje Autonomo 2.py
- corazon.png (debe estar en la misma carpeta)
